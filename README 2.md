## descripTion de la procedure d'installation XAMPP

J'ai commencée par installer XAMPP.Ensuite, j'ai activé Apache Web server puis MySQL Database, puis attendu que les deux sont activent (vert). J'ai ouvert dans le dossier de XAMPP,le dossier httpdocs où j'ai déposée ( déziper) le dossier contenant mon site (sae105). Ensuite je lance l'URL sur le navigateur: http://localhost/sae105/.
